Как это работает

Три клика от фото в телефоне до отчёта на столе у руководителя.

  1. 1

    Перетащите изображение

    JPG, PNG или TIFF — с микроскопа, телефона или сканера. Обработайте сразу целую папку в пакетном режиме.

  2. 2

    Нажмите «Анализ»

    Под капотом — Cellpose. Готовые модели находят все колонии без настройки параметров. Каждой присваивается ID, площадь, диаметр, эксцентриситет.

  3. 3

    Выгрузите отчёт

    PDF на одну страницу: превью, гистограмма, таблица. CSV для Excel. JSON-манифест с SHA-256 и параметрами запуска — для воспроизводимости в публикациях.

Возможности

📥

Перетаскивание файлов

Сбросьте JPG, PNG или TIFF прямо из проводника. В пакетном режиме обрабатывает сразу всю папку с десятками чашек.

🔬

ИИ-сегментация

На базе библиотеки Cellpose. Предобученные модели находят колонии разной формы и размера без ручной настройки.

✏️

Ручная коррекция

Один клик удаляет ошибочно найденную колонию. Кисть добавляет пропущенную. Метрики пересчитываются мгновенно.

🔍

Зум и панорама

Колесо мыши — приближение в любую точку, Space + перетаскивание — сдвиг. F — вписать в окно. Удобно для деталей.

📏

Реальные единицы

Укажите микрометры на пиксель — и площади выводятся в мкм², диаметры в мкм. Готово для раздела «Методы» в статье.

📄

Отчёты для публикаций

PDF-отчёт с превью, гистограммой и таблицей. CSV для электронных таблиц. JSON-манифест с SHA-256 для воспроизводимости.

💻

Работает на ноутбуке

Windows, macOS, Linux. CUDA — если есть, обычный CPU — если нет. Данные не покидают рабочую станцию.

🔁

Пакетная обработка

Указали папку — получили CSV, PDF и JSON для каждого изображения плюс сводный summary.csv. Можно идти пить чай.

🔒

Конфиденциально

Нет аналитики, нет аккаунтов, нет загрузки в облако. Чувствительные данные пациентов и фармы остаются у вас локально.
